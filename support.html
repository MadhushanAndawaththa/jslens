Support

Everything you need to install, troubleshoot, and report bugs.

Getting started

Job Search Lens runs entirely inside Chrome. Open linkedin.com/jobs for the full set of features: card dimming, inline company stats, and keyword highlights. The keyword highlighter also works on any other site you visit (Indeed, Glassdoor, Seek, company career pages, etc.) so your saved terms travel with you.

  1. Install from the Chrome Web Store.
  2. Click the Job Search Lens icon in your browser toolbar.
  3. Add a keyword in the popup or right-click selected text on a page → Add to Highlighter.
  4. Open LinkedIn Jobs and watch viewed / saved / applied cards fade automatically.

Troubleshooting

  • If the popup shows "Site access needed", enable extension access for the current site in Chrome's extension settings, then reload the tab.
  • If keyword matches are missing, confirm the term is actually visible on the page and that Pause all is off in the popup.
  • If LinkedIn card dimming stops working after a LinkedIn update, check for a newer extension version — LinkedIn changes its DOM frequently and the extension occasionally needs a patch to keep up.
  • If the popup looks broken or the wrong theme flashes briefly, hard-reload the popup (close and reopen). Theme is persisted in localStorage.

Keyboard shortcuts

  • Ctrl/Cmd + Enter inside the keyword input — submit and add the keyword.
  • Escape while a color picker is open — close the popover and return focus to the swatch button.
  • Right-click any selected text on any page — Add "[text]" to Highlighter.

Report a bug or request a feature

Use the issue tracker. Please include:

  • Chrome version and OS.
  • Extension version (shown in chrome://extensions).
  • The page or URL where the issue happened (or a description if it's private).
  • What you expected vs. what happened.
