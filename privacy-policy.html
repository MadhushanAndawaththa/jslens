Privacy Policy

Last updated: January 2026

Job Search Lens is a local-only Chrome extension built for LinkedIn Jobs. This page explains what the extension accesses, how that information is used, and what is not collected or shared.

What the extension does

On LinkedIn Jobs, Job Search Lens passively dims job cards that LinkedIn itself labels as Viewed, Saved, or Applied, and shows inline company size and follower counts near job titles. It also highlights any keywords you've saved — on LinkedIn pages and, as a portable bonus, on every other website you visit.

What data is stored

  • Keywords you save.
  • Color choices for those keywords.
  • Extension settings such as pause state, dim-state toggles, and theme preference.

These values are stored locally in chrome.storage.local on your device and are never transmitted anywhere.

What data is not collected

  • No account information is collected.
  • No personal data is transmitted to the developer or any third party.
  • No job application data is uploaded to any server.
  • No analytics, telemetry, tracking pixels, or third-party advertising is used.
  • No remote code is executed.

How page content is used

On LinkedIn Jobs, the extension reads the Viewed, Saved, and Applied labels LinkedIn already renders on each card, plus the company size and follower-count text LinkedIn renders inline. On any other website, the extension reads page text locally to find matches for the keywords you've saved.

This processing is on-device only. The extension does not send page content, browsing history, or any metadata to any external endpoint.

Permissions

  • contextMenus — add selected text as a keyword via right-click on any page.
  • storage — persist your keywords and settings locally in the browser.
  • activeTab — let the popup talk to the currently active tab for status and match navigation when you open the popup.
  • http://*/* and https://*/* — required for keyword highlighting to work on every site you visit. LinkedIn-specific dimming and company stats remain scoped to LinkedIn Jobs.

Data sharing

No stored or processed data is sold, rented, transferred, or disclosed to any third party.

How we keep this promise

The extension package shipped to the Chrome Web Store contains only the JavaScript, CSS, and image files needed to run locally. You can inspect everything that runs in your browser via chrome://extensions → Inspect views. There is no remote loader, no analytics SDK, and no network call to a developer-controlled server.

Contact

For support or privacy questions, visit the support page or open an issue on the issue tracker.
